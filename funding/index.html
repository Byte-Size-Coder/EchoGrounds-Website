Funding Support

Funding options for First Nations language revitalization projects

Many First Nations communities across Canada use federal language revitalization funding to support digital tools like EchoGrounds. These programs are designed to help communities reclaim, strengthen, and teach their languages, and software subscriptions are eligible expenses.

1. Indigenous Languages and Cultures Program (ILCP)

Indigenous Languages Component (ILC)

The ILCP is the primary federal program supporting First Nations language revitalization. Communities can use this funding to cover:

  • Digital language tools
  • Software subscriptions
  • Language learning platforms
  • Documentation and archiving
  • Community-led language projects

Who can apply: First Nations, Inuit, Metis communities, tribal councils, cultural centres, and other eligible organizations.

Learn more: Search for "Indigenous Languages and Cultures Program (ILCP) - Canada" on the Government of Canada website.

2. First Nations Languages Funding Model

This is a multi-year funding stream under the ILCP that provides stable support for long-term language revitalization efforts. It is ideal for communities planning multi-year EchoGrounds subscriptions.

Supports:

  • Multi-year language strategies
  • Immersion programs
  • Digital tools and software
  • Community language planning

Who can apply: First Nations communities, First Nations organizations, tribal councils, and cultural education centres.

Learn more: Search for "First Nations Languages Funding Model - Canada" on the Government of Canada website.

3. Office of the Commissioner of Indigenous Languages (OCIL)

While OCIL does not provide direct funding, it offers guidance and resources that strengthen community language plans and funding applications. Communities can reference OCIL's mandate to justify the need for digital tools like EchoGrounds.

Supports:

  • Language rights
  • Community planning
  • Best practices for revitalization
  • Strengthening funding proposals

Learn more: Search for "Office of the Commissioner of Indigenous Languages (OCIL)" on the Government of Canada website.

How These Programs Support EchoGrounds

EchoGrounds is eligible for funding because it directly supports:

  • Language learning and teaching
  • Documentation and preservation
  • Intergenerational knowledge transfer
  • Community-controlled digital resources
  • Long-term language sustainability

Communities can include the EchoGrounds subscription as a project expense in their funding application.
